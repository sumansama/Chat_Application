# 💬 Real-Time Chat Application

A real-time web-based chat platform built using **HTML, CSS, and JavaScript** with WebSockets. Users can join or create chat rooms, send messages in real time, and enjoy a smooth, interactive messaging experience.

---

## 📌 Features

- 🧑‍💼 **User Authentication**
  - Choose a unique username before entering a chat room.
  - Prevents duplicate usernames.

- 🛋️ **Chat Rooms**
  - Join existing chat rooms or create your own.
  - See a list of available rooms dynamically.

- ✉️ **Messaging System**
  - Real-time text messaging using WebSockets.
  - See sender name, timestamps, and formatted text (bold, italic, links).
  - Auto-scrolls to latest messages.

- 🧠 **User Experience**
  - Responsive UI for desktop and mobile.
  - Notifications for new messages.
  - Handles edge cases like empty input or username conflicts.




## 🛠️ Technologies Used

- **HTML** – For structure and layout  
- **CSS** – For styling and responsive design  
- **JavaScript (ES6)** – For interactivity and WebSocket communication

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, etc.)


### Running the Project

#### Option 1: Without Backend (Frontend UI Demo Only)

1. Unzip the project folder.
2. Open `index.html` in your browser.
3. You can interact with the UI, but real-time chat won’t function without a server.
